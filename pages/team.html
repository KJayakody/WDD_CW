Working Progress

To track our work efficiently, we used Jira for project management, allowing us to organize and monitor tasks effectively. Our team divided responsibilities, ensuring that each member focused on developing specific pages. This approach streamlined our workflow and kept progress on track. Before development, we created a detailed prototype using Figma, which helped visualize the final design and user experience. By following this structured process, we ensured smooth collaboration and efficient execution of our project goals.

Collaberation

For collaboration, we used GitHub to manage our code efficiently. Each team member worked on separate branches, making contributions and reviewing changes through pull requests. This streamlined our workflow and ensured version control. By using GitHub, we maintained a smooth development process, resolving conflicts and merging updates seamlessly.
