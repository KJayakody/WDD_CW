Volunteer Page

Technical Description

HTML Structure

The volunteer page employs a user-centered approach with interactive elements that allow visitors to discover volunteer opportunities. Key structural elements include:

  • Filter System: Implemented search input and category dropdown for filtering volunteer programs
  • Program Cards: Structured display of volunteer opportunities with consistent information architecture
  • Review Section: Showcases user feedback with a summary and individual review cards
  • Interactive Form: Includes emoji-based rating system and feedback submission functionality

CSS Techniques

The styling for the Volunteer page employs several advanced CSS techniques:

  • Background Handling: Full-page background image with fixed positioning and overlay
  • Card Design: Consistent card components with hover effects for interactive feedback
  • Flexbox Layouts: Used for organizing program cards, review sections, and rating buttons
  • Interactive Elements: Custom styling for form elements, buttons, and rating emojis
  • Color Transparency: Used rgba() color values to create semi-transparent backgrounds
  • Custom Typography: Applied font styling to create visual hierarchy
  • Border Radius: Consistently applied rounded corners for a cohesive design language
  • Box Shadows: Added subtle shadows to create depth and emphasize important elements

JavaScript Implementation

I developed JavaScript functionality for volunteer page to enhance interactivity with the user (Filtering System, Rating System)

  • Program Filtering System
    • Created event listeners for the search input and category dropdown
    • Implemented filtering logic based on program names and categories
    • Applied dynamic display changes based on filter criteria
  • Rating System
    • Developed an emoji-based rating selection with visual feedback
    • Added validation to ensure ratings are selected before submission
    • Implemented popup feedback system

Content Page

The Importance of Oceans for the Planet

Technical Description

HTML Structure

For the content page about ocean importance, I implemented a structured layout using semantic HTML5 elements to enhance accessibility and SEO. Key HTML elements include:

  • <section> tags to divide content into meaningful sections (hero, introduction, information, challenges, conclusion)
  • Hero section with a compelling background image and call-to-action button
  • Information boxes using flexbox containers to present environmental, economic, and social importance of oceans
  • Strategic use of headings (<h1>through<h1>) to create a logical document outline
  • SVG icons for interactive elements like the scroll-to-top button

CSS Techniques

The styling for the content page employs several advanced CSS techniques:

  • Responsive Design: Applied flexible layouts that adapt to different screen sizes
  • CSS Variables: Used for consistent color themes (primary and secondary colors)
  • Pseudo-elements: Created decorative elements like the underline after section headings using ::after
  • Flexbox: Implemented for layout organization, particularly in the information boxes section
  • CSS Positioning: Used relative and absolute positioning for elements like the hero overlay and scroll button
  • Image Handling: Applied object-fit: cover to ensure images maintain proper proportions
  • Transitions: Added subtle animations for interactive elements like the related content boxes
  • Linear Gradients: Created overlay effects on the hero section and content boxes

JavaScript Implementation

I developed JavaScript functionality for content page to enhance interactivity with the user(Scroll-to-top Functionaity)

  • Created a smooth scrolling effect using the Window.scrollTo() method
  • Applied event handling for button clicks
