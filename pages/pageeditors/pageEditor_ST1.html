Splash Page

Technical Description

Technical Description

HTML Structure

The splash screen uses a straightforward but effective HTML setup:

  • It has all the proper HTML5 basics - DOCTYPE declaration, meta tags for character encoding and responsive design.
  • It pulls in the "Protest Strike" font from Google Fonts, which gives the page its unique typography.
  • The content is organized in a simple div container that holds
    • A big headline for "LIFE BELOW WATER"
    • A short mission statement paragraph
    • A list of team members' names
    • A loading animation (using a GIF)
  • There's a simple JavaScript function at the bottom that automatically redirects users to the home page after 5 seconds, so they don't have to click anything.

CSS Techniques

The CSS does most of the heavy lifting to make the splash screen visually appealing:

  • It uses a full-screen ocean background image that fills the entire viewport.
  • Everything is centered using flexbox (display: flex with align-items and justify-content set to center).
  • The title is huge (100px) with text shadows to make it pop against the background.
  • The mission statement has a different color (#1e3d4e) to create visual contrast./li>
  • The team members' names are displayed in a horizontal row at the bottom of the screen using flexbox again, with absolute positioning to keep them at the bottom.
  • Text has shadows applied to improve readability against the background image.
  • The loading spinner is sized appropriately (125px) with some top margin to separate it from other elements.

JavaScript Implementation

The JavaScript is minimal but serves an important purpose:

  • It uses the setTimeout() function to create a 5-second delay before automatically redirecting the user.
  • After the delay, it changes the window.location.href to "home.html", which takes the user to the main page.
  • This creates a smooth transition from the splash screen to the actual content without requiring any user input.
  • The timing gives users enough time to see the project title, mission statement, and team members before moving on.
All these techniques work together to create a visually striking intro screen that sets the tone for the website while the main content loads.

Content Page

The Importance of Oceans for the Planet

Technical Description

HTML Structure

For the content page about ocean importance, I implemented a structured layout using semantic HTML5 elements to enhance accessibility and SEO. Key HTML elements include:

  • <section> tags to divide content into meaningful sections (hero, introduction, information, challenges, conclusion)
  • Hero section with a compelling background image and call-to-action button
  • Information boxes using flexbox containers to present environmental, economic, and social importance of oceans
  • Strategic use of headings (<h1>through<h1>) to create a logical document outline
  • SVG icons for interactive elements like the scroll-to-top button

CSS Techniques

The styling for the content page employs several advanced CSS techniques:

  • Responsive Design: Applied flexible layouts that adapt to different screen sizes
  • CSS Variables: Used for consistent color themes (primary and secondary colors)
  • Pseudo-elements: Created decorative elements like the underline after section headings using ::after
  • Flexbox: Implemented for layout organization, particularly in the information boxes section
  • CSS Positioning: Used relative and absolute positioning for elements like the hero overlay and scroll button
  • Image Handling: Applied object-fit: cover to ensure images maintain proper proportions
  • Transitions: Added subtle animations for interactive elements like the related content boxes
  • Linear Gradients: Created overlay effects on the hero section and content boxes
