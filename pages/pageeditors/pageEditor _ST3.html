Team Page

Technical Description

  • The team page is divided into two section and structured using display: flex; and flex-direction: column;.
  • Set a backgroung image to body using CSS. 
                    
                    background: url(../assets/team.bg.jpg);
  • Created image cards with thumbnails using div class="card" and placed inside the section class="card-container". To scale up when hover above the card added transform: scale(1.1) to CSS. This scale the element without moving the other elements.
  • To create the effects (exapnd and transparency) on cards when selected, used two seperate CSS classes selected and not-selected. Using JS element.classList.toggle("class") add or remove the class when card selected or not.
  • To show/ hide extra details in cards, used two seperate classes show with display: block; and hide with display: none;.

    Challenges and Lessons Learned

    Designing a basic structure for the web page
    It challenging to develop the web pages without having a clear design to follow. For resolve this we used Figma(Visit design) to design the web page before statrt coding.
    Layouting multiple elements
    Its difficult to position multiple elements only using margin properties. By using position CSS like flex and grid It is easy to position elements.
    Keeping track with the other members
    When each member developing the pages remotly, it's hard to keep track on pages and testing the web site. We used GitHub (Visit Repository) to share and review our changes. We also joined weekly sheduled online meetings using Google Meet to overview the progrees and palning the future tasks. To keep all this managed and tracked we used Jira Project Mangement.

    In this experience I am able to learn about how to develop a project while collaborating with others, basic web developing. This also helped to learn tools like Figma, GitHub, Jira etc.
